# Kotlin-Android-GreenDao
A demo Application to show the usage of Green Dao using kotlin.

Steps Followed:
1. Created A project using kotlin
2. Created A Android library module using java 
3. Added the Green dao library to the module
4. Created the necessary Db classes in the library using java
5. Imported the module in to the android project.
6. Now the library module generates all the necessary dao classes and provides then to the kotlin project.
